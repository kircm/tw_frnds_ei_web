This web-app may be used as an utility for your Twitter account to:

  1. Export all the accounts you follow into a CSV (comma-delimited) file for you to keep as a back-up for later use.
  2. Import all the accounts present in a CSV file as "friends" into Twitter, that is, follow all those accounts.

For the app to be able to do that it will need to authenticate you into Twitter and authorize this app to perform actions on your behalf.

The app uses Twitter's API. The only operations this app does are:

  • Query friends account names and ID's: friends/list.
  • Perform friendships/create requests.

This app uses the tw_frnds_ei program to interact with Twitter. The program takes care of several things like:

  • Loop through cursors (chunks of data) when twitter API breaks the data in chunks.
  • Make sure it doesn't exceed Twitter API's rate limits.
